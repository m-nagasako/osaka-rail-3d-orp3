## 5. 設計書からの変更・確定事項

1. 深さの持ち方: stations.elevM(駅単位)→ **lines.elev[]（路線×駅）**。乗換駅の立体交差表現のため
2. avgSpeedKmhの意味: 表定速度 → **駅間走行速度**(停車時間の二重計上を解消)
3. 倍速プリセット: {×1, ×10, ×60, ×300} で確定
4. 外部設計F2のうち垂直フィルタはM2へ(承認済)
5. 北大阪急行南北線は別事業者レイヤとして収録。江坂は御堂筋線と共有し、直通運転は表現しない
6. ランドマークは装飾レイヤに統合し、海岸線・淀川と同じトグルでON/OFFする
7. JR西日本は大阪市内+市境1駅の範囲で9路線を収録。おおさか東線は全7駅収録
8. 垂直フィルタは地下/地上/高架の3分類。路線チューブは駅間セグメント単位で切替

## 6. 既知の制限(UIのクレジットにも自動表示)

運行は運転間隔ベースの近似(実ダイヤではない)/ 線形は駅間の滑らか補間(実線形ではない)/ 深さ・高さは概算・縦14倍誇張(値は `src/config.js` の EXAGGERATION を正とする) / 直通運転なし / 北大阪急行の駅別乗降人員は未投入(注記表示) / JR乗降人員は乗車人員×2換算で、上位50駅に無い駅は注記表示 / ランドマークは装飾用の概略座標 / 中央線夢洲区間の半数運行は未対応 / 駅座標は2017年系譜データ+夢洲・北急延伸駅手動パッチ(公開時はN02差し替え推奨)
