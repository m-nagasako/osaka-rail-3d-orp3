## 1. 実施内容

Sprint4指示書に基づき、私鉄5事業者のM3データを追加した。

- 阪急: 神戸本線、宝塚本線、京都本線、千里線
- 阪神: 本線、阪神なんば線
- 京阪: 京阪本線、中之島線
- 近鉄: 難波線、大阪線、南大阪線
- 南海: 南海本線、高野線

近鉄奈良線は「布施止まり」指定だが、布施1駅のみでは路線として成立せず、チューブ描画・列車スケジュール・駅間距離検証の対象にできないため、指示書の例外扱いに沿ってSprint4では非収録とした。布施は近鉄大阪線の境界駅として収録済み。

## 2. データ設計

- 入力側の`scripts/build-data.js`と`scripts/patches/*.json`を更新し、`public/data/*.json`は`npm run build-data`で生成した
- 主要乗換駅は名前ベース共有ルールを追加し、大阪難波/京橋/西九条/新今宮/天下茶屋などを1ノード化した
- 大阪と梅田、難波/なんば/大阪難波/JR難波、大阪阿部野橋と天王寺は分離維持した
- 私鉄の深さ/高さ、ラインカラー、運転間隔はSprint4時点の概算値としてメタ情報と注意書きに反映した
- 私鉄単独駅の乗降人員は未投入注記を表示する。Metro/JR共有駅は既存出典値を継続表示する
